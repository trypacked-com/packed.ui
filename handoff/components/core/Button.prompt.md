Packed's primary action button — warm, rounded, with a soft brand glow on the primary variant. Use for the single most important action in a view.

```jsx
<Button variant="primary" size="md" iconLeft={<PlusIcon/>}>Add flight</Button>
```

Variants: `primary` (orange, glowing), `secondary` (white, bordered), `ghost` (text-only brand), `danger`. Sizes `sm | md | lg`. Set `pill` for fully-rounded, `fullWidth` to stretch, `as="a"` for links.
